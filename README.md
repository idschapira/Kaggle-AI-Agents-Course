# Kaggle 5-Day AI Agents — Diário de Aprendizado

Repositório de estudos do curso **5-Day AI Agents** (Kaggle), com foco em fundamentos de Engenharia de Dados e construção de agentes de IA.

## Estrutura do projeto

```
.
├── apps/           # Apps criados no Google AI Studio (código exportado)
├── notebooks/      # Jupyter notebooks com exercícios e experimentos
├── data/
│   ├── raw/        # Dados originais, não tratados
│   └── processed/  # Dados tratados/transformados
├── scripts/        # Scripts Python reutilizáveis (.py)
├── docs/           # Whitepapers e anotações de apoio
└── Diário_Aprendizado.md   # Registro de progresso, sessão a sessão
```
